## 📋 Template Usage Guide

### Quick Start

```typescript
import { TemplateGenerator } from '@/lib/templates/generator';
import { getTemplatesForIndustry } from '@/lib/templates/configs';

// Get recommended templates for industry
const templates = getTemplatesForIndustry('healthcare');
// Returns: ['healthcare-01']

// Generate page
<TemplateGenerator
  templateId="healthcare-01"
  business={{
    businessName: "City Medical Center",
    industry: "healthcare",
    coreOffer: "Comprehensive primary care",
    targetCustomer: "Families",
    differentiator: "Same-day appointments",
  }}
  generatedContent={aiGeneratedContent}
  heroImageUrl="/hero.jpg"
/>
```
